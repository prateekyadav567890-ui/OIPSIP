## 📖 Project Description

This is a Java console-based ATM Interface developed using Object-Oriented Programming (OOP). The application simulates basic ATM operations such as user authentication, balance management, deposits, withdrawals, fund transfers, and transaction history. The system also provides login security with limited attempts and maintains transaction records for each session.

---

## ✨ Features

- Secure User ID and PIN authentication
- Maximum 3 login attempts
- View Transaction History
- Deposit Money
- Withdraw Money
- Transfer Money between accounts
- Insufficient balance validation
- Transaction history stored using ArrayList
- Multiple user accounts
- Recursive menu navigation
- Object-Oriented Programming (OOP) implementation

## ▶️ How to Run

1. Open the project in any Java IDE (Eclipse, IntelliJ IDEA, VS Code, etc.).
2. Compile the program.
3. Run the `ATMInterface.java` file.
4. Log in using a valid User ID and PIN.
5. Choose any ATM operation from the menu.
6. Perform transactions until you decide to quit.
