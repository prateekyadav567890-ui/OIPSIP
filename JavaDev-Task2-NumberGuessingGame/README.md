## 📖 Project Description

This is a Java console-based Number Guessing Game developed using Core Java. The computer generates a random number, and the player attempts to guess it within a limited number of attempts. The game provides hints after every guess and supports multiple difficulty levels with a play-again option.

---

## ✨ Features

- Random number generation
- Three difficulty levels:
  - Easy (1–50, 10 attempts)
  - Medium (1–100, 7 attempts)
  - Hard (1–200, 5 attempts)
- "Too High" and "Too Low" hints
- Maximum attempt limit
- Displays the correct number after losing
- Round summary with attempts taken
- Multiple rounds support
- Play Again option
- User-friendly console interface

## ▶️ How to Run

1. Open the project in any Java IDE (Eclipse, IntelliJ IDEA, VS Code, etc.).
2. Compile the program.
3. Run the `NumberGuessingGame.java` file.
4. Select a difficulty level.
5. Guess the number within the allowed attempts.
6. Choose whether to play another round.
